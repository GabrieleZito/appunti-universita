Creazione Database

```sql
CREATE DATABASE nome_db
```

Creazione tabelle

```sql
CREATE TABLE nome_tabella(
	nome_colonna [tipo] [default] [vincoli]
	...
	
	vincoli di tabella 
)
```

Visualizzare schema tabella

```sql
DESCRIBE nome_tabella
```

Selezionare il database su cui lavorare

```sql
USE nome_db
```

### Domini elementari

- CHAR(n): stringa di $n$ caratteri
- VARCHAR(n): stringa fino a $n$ caratteri
- NUMERIC(precisione, scala)
- DEC(precisione, scala)
- INT
- TINYINT, SMALLINT,MEDINT,BIGINT:    1, 2, 3, e 8 byte
- FLOAT(precisione)
- DOUBLE
- DATE: formato aaaa-mm-gg
- TIME: formato hh-mm-ss
- TIMESTAMP

Precisione → numero massimo di cifre, sia intere che decimali

Scala → numero di cifre dopo la virgola

### Domini Custom

```sql
CREATE DOMAIN Voto
AS SMALLINT DEFAULT NULL
CHECK (value >= 18 AND value <=30)
```

### Vincoli intrarelazionali

```sql
colonna NOT NULL
colonna UNIQUE
colonna PRIMARY KEY
colonna CHECK ...
```

### Vincoli interrelazionali

```sql
CHECK
colonna REFERENCES tabella(colonna)
oppure
FOREIGN KEY (colonna) REFERENCES tabella(colonna)
```

### Reazione alla violazione

```sql
FOREIGN KEY(colonna) REFERENCES tabella(colonna) ON DELETE reazione
FOREIGN KEY(colonna) REFERENCES tabella(colonna) ON UPDATE reazione
```

Reazioni:

- NO ACTION - (default) impedisce la cancellazione/modifica
- CASCADE - cancella/modifica a cascata le righe referenti
- SET NULL - assegna il valore null
- SET DEFAULT - assegna il valore di default
