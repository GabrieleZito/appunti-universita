Albero binario con le proprietà:

- completo fino al penultimo livello
- ogni nodo memorizza una chiave
- la chiave del padre è maggiore della chiave del figlio

### Altezza Heap

Un heap con $n$ nodi ha altezza $\Theta(\log n)$

Dimostrazione:

$h$ i livelli dell’heap, i primi $h-1$ sono completi $\Rightarrow$ il numero di nodi è $\sum\limits_{i=0}^{h-1}2^i=2^h-1$

Il numero totale di nodi sarà $2^h-1\le n\le2^{h+1}-1$

Passando ai logaritmi

$$\begin{cases}
h\le\log(n+1)=O(\log n)\\
h+1\ge\log(n+1)=\Omega(\log n)
\end{cases}\Rightarrow \Theta(\log n)$$

### Cancellazione massimo

Nell’heap il max si trova nella radice. Cancellata la radice si sostituisce con l’ultima foglia e si ripristina l’heap con $\text{fixHeap()}$

### FixHeap

```jsx
function FixHeap(nodo v, heap H)
	IF v è una foglia THEN
		return
	u <- figlio di v con chiave massima
	IF chiave(v) < chiave(u) THEN
		scambia chiave(v) e chiave(u) // scambia solo le chiavi
		Fixheap(u, H)
```

### Heapify - Costruzione dell’Heap

```jsx
function Heapify(albero T)
	IF T vuoto THEN
		return
	Heapify(sottoalbero sx di T)
	Heapify(sottoalbero dx di T)
	FixHeap(radice(T), T)
```

$$
T(n)=2T(\frac n2)+\log n=O(n)\quad \text{per il caso 1 del Th. Master}
$$

## Heap Sort

```jsx
function HeapSort(array A)
	H = Heapify(A)
	x <- []
	WHILE NOT H.isEmpty() DO
		prendi il max(H) e aggiungi in coda ad x
		fixHeap(H)
	return x
```

## Merge Sort

Divide et Impera

- dividi l’array a metà
- ordina il sotto array

### Merge - Unione di sequenze ordinate

```jsx
function Merge(Array A, int i1, int f1, int f2)
	X <- array di dimensione f2-i1+1
	i <- 1
	i2 <- f1+1
	WHILE i1 <= f1 AND i2 <= f2 DO
		IF A[i1] <= A[i2] THEN          // <- nel caso peggiore si effettuano |x|-1 confronti
			X[i] <- A[i1]
			i++
			i1++
		ELSE
			X[i] <- A[i2]
			i++
			i2++
	IF i1 < f1 THEN
		copia A[i1...f1] alla fine di X
	ELSE
		copia A[i2...f2] alla fine di X
	copia X in A[i1...f2]
```

### Mergesort

```jsx
function MergeSort(Array A, int i, int f)
	IF i >= f THEN
		return
	m <- (i+f)/2
	MergeSort(i, m)
	MergeSort(m+1, f)
	Merge(A, i, m, f)
```

### Analisi

$$
T(n)=2T(\frac n2)+O(n)
$$

$$
T(n) = O(n\log n)\text{ per il caso 2 del Th. Master}
$$

## Quick Sort

Migliora il merge sort aggiungendo un controllo nella divisione dell’array

### Partition

Riordina l’array in base al perno

```jsx
function Partition(array A, int i, int f)
	x <- A[i]
	inf <- i
	sup <- f+1
	WHILE true DO
		DO
			inf++
		WHILE inf <= f AND A[inf] <= x
		DO
			sup--
		WHILE A[sup] >= x
		IF inf < sup THEN
			scambia A[inf] e A[sup]
		ELSE 
			break
	scambia A[i] e A[sup]
```

### Quicksort

```jsx
function QuickSort(array A, int i, int f)
	IF i >= f THEN
		return
	m <- Partition(A, i, f)
	QuickSort(A, i, m-1)
	QuickSort(A, m+1, f)
```

### Analisi

La partizione costa $n-1$

$T(n)=T(a)+T(b)+n-1$, nel caso peggiore si ha $a=0,b=n-1$ o viceversa

$$
T(n)=T(n-1)+n-1\Rightarrow T(n)=O(n^2)\text{ con il metodo dell'iterazione}
$$

### Randomizzazione

Per rendere l’algoritmo più efficiente si randomizza la scelta del perno

$$\begin{gathered}
T(n)=\sum\limits_{a=0}^{n-1}\bigg(\frac1n\big(n-1+T(a)+T(n-1-a)\big)\bigg)=\\
=\frac{n-1}{n}\sum\limits_{a=0}^{n-1}1+\frac1n\sum\limits_{a=0}^{n-1}T(a)+\frac1n\sum\limits_{a=0}^{n-1}T(n-1-a)=\\
=\frac{n-1}{n}+\frac1n\sum\limits_{a=0}^{n-1}T(a)+\frac 1n\sum\limits_{a'=0}^{n-1}T(a')=n-1+\frac2n\sum\limits_{a=0}^{n-1}T(a)
\end{gathered}
$$

che ha soluzione $T(n)\le 2n\log n$

Dimostrazione per induzione che $T(n)\le\alpha n\ln n$

base → $n=1$         $T(n)=0\Rightarrow0\le\alpha(1\ln1)\quad\forall\alpha$

passo → supponiamo vero che $\exists\ \alpha \mid T(i)\le\alpha(i\ln i)\text{ per }i<n$

dimostro per $n$

$$\begin{gathered}
T(n)=n-1+\frac2n\sum\limits_{i=0}^{n-1}T(i)\le n-1+\frac2n\sum\limits_{i=0}^{n-1}\alpha i\ln i=\\
=n-1+\frac {2\alpha}n\sum\limits_{i=2}^{n-1}i\ln i\le n-1+\frac{2\alpha}n\int\limits_2^nx\ln x\ dx
\end{gathered}
$$


> [!info] In generale
> $$
\sum\limits_{i=a}^bf(i)\le\int\limits_a^{b+1}f(x)dx
$$


$$
\begin{gathered}
T(n)\le n-1+\frac{2\alpha}n\int\limits_a^nx\ln x\ dx=n-1+\frac{2\alpha}{n}\bigg\{ \big[\frac{x^2\ln x}{2}\big]^n_2 - \int\limits_2^n\frac x2\ dx\bigg\}=\\
=n-1+\frac{2\alpha}{n}\bigg\{  
\frac{n^2\ln n}{2}-\frac{n^2}{4}-\frac{4\ln 2}{2}+\frac44
\bigg\}=n-1+\alpha n\ln n-\alpha\frac n2-\cos t\le\\
\le\alpha n\ln n
\end{gathered}
$$

è vera se $n-1<\alpha\frac n2$ che è verificata $\forall\alpha\ge2$

Quindi $T(n)\le2n\log n=O(n\log n)$

## Altri tipi di ordinamento

Se in un algoritmo non sono presenti confronti, o sono ammesse altre operazioni, bisogna passare ad un altro modello

## Integer Sort

In un array di interi conta quante volte appare un numero e restituisce una sequenza ordinata

### $n$  interi in $[1, n]$

```jsx
function IntegerSort(array A, int n)
	Y <- array vuoto di dimensione n
	FOR i = 1 TO n                 ┐  n
		Y[i] <- 0                  ┘
	FOR i = 1 TO n                 ┐  2n               
		Y[A[i]]++                  ┘
	j <- 1
	FOR i = 1 TO n                 ┐
		WHILE Y[i] > 0             |
			A[j] <- i              |  3n
			j++                    |
			Y[i]--                 ┘
		
```

$$
6n = O(n)
$$

### $n$ interi in $[1,k]$

```jsx
function IntegerSort(array A, int n, int k)
	Y <- array vuoto di dimensione k
	FOR i = 1 to k                ┐  k
		Y[i]=0                    ┘
	FOR i = 1 to n                ┐  2n
		Y[A[i]]++                 ┘
	j <- 1
	FOR i = 1 to k                ┐
		WHILE Y[i] > 0            |
			A[j] <- i             |  k+2n
			j++                   |
			Y[i]--                ┘
```

$$
O(n+k)
$$

## Radix Sort

Ordina gli elementi partendo dalla cifra meno significativa a quella più significativa chiamando il bucket sort

### Analisi

Il Bucket Sort costa $O(n)$ e viene chiamato $\log_{10}k$ volte

Il costo totale è $O(n\log k)$

### Teorema

Usando una base $b = \Theta(n)$ l’algoritmo ordina $n$ interi $\in[1,k]$ in $O(n(1+\frac{\log k}{\log n}))$

Dim.

BucketSort = $O(n)$, chiamato $O(\log_n k)$ volte

- se $k>n,\ T(n)=O(n\log_nk)=O(n\frac{\log k}{\log n})$
- se $k<n,\ T(n) = O(n)$

Quindi conmbinati

$$
T(n)=O(n(1+\frac{\log k}{\log n}))
