
Organizzazione gerarchica

- dati all’interno di nodi
- relazioni rappresentate da archi

Coppia $T=(N, A)$, con $N$ un insieme di nodi e $A$ di archi, $A\subseteq N\times N$

Ogni nodo ha un solo genitore, e può avere nessuno o più figli

Nodi foglia → Nodi che non hanno figli

Profondità → Numero di archi da attraversare per raggiungere un nodo partendo dalla radice

Altezza → Profondità massima

Grado → Numero di figli

Albero d-ario → Albero in cui ogni nodo ha d figli

## Rappresentazione degli alberi

### Vettore padri (Array)

Array che contiene i campi *Info* e *Parent.* Info contiene i dati, Parent contiene l’indice della cella del padre.

### Vettore posizionale (Array)

Per gli alberi d-ari. Ogni nodo ha una posizione prestabilita nell’array.

L’i-esimo figlio di $v$ si trova in posizione $P[d*v+i]$

Il padre si trova in $\begin{cases}\lfloor\frac vd\rfloor,\qquad \text{se}\ \  v\ \text{mod}\ d\ne0\\\lfloor\frac vd\rfloor-1,\quad \text{altrimenti}\end{cases}$

### Puntatori ai figli (Liste)

Lista di nodi in cui ogni nodo ha i dati e i puntatori ai figli

### Liste di puntatori ai figli (Liste)

Lista in cui ogni nodo ha una lista di puntatori ai figli

### Primo figlio - Fratello successivo

Ogni nodo ha il puntatore ad un figlio ed uno ad il fratello

## Visita agli alberi - $O(n)$

### Visita in profondità - DFS - Depth First Search

CON PILA

```jsx
function VISITDFS(nodo r)
	Pila S
	S.push(r)
	WHILE NOT S.isEmpty() DO
		U <- S.pop()
		IF U != null THEN
			visita U
			FOR h IN figli di U DO
				P.enqueue(h)
```

RICORSIVO

```jsx
function VISITDFS(nodo r)
	IF r = null THEN
		RETURN
	visita nodo r
	FOR h IN figli di r DO
		VISITDFS(h)
```

### Visita in ampiezza - BFS - Breadth First Search

CON CODA

```jsx
function BFS(nodo r)
	Coda P
	P.enqueue(r)
	WHILE NOT P.isEmpty() DO
		U <- P.dequeue()
		IF U != null THEN
			visita U
			FOR h IN figli di U DO
				P.enqueue()
```

### Teorema - Altezza albero binario

Sia $T$ albero binario, ogni nodo ha 2 figli, $k$  il numero di foglie. L’altezza dell’albero è almeno $\log_2 k$

Dimostrazione per induzione

BASE. $k=1$, quindi un albero con solo la radice

$\log_21 = 0$

PASSO.  Supponiamo vera $h(i)\ge\log_2i$, con $i< k$

L’albero avrà altezza almeno $h(k)\ge1+h(\frac k2)=1+\log_2\frac k2=1+\log_2k-\log_22=\log_2 k$

## Ordinamento

### Teorema

Il numero di confronti necessari per ordinare $n$ elementi è, nel caso peggiore, $\Omega(n\log n)$

Dim.

- Il numero di foglie in un albero di decisione è $n!$
- Il numero di confronti fatti è, nel caso peggiore l’altezza dell’albero
- $h(n!)\ge\log_2 n!$
- $n!\approx\sqrt{2\pi n}(\frac n2)^n\quad\Rightarrow\quad\log n!\approx n\log n-n\log e+\frac12\log n+\frac12\log2\pi$
- Il numero di confronti è $n\log n-O(n)$

## Selection Sort - $O(n^2)$

```jsx
function SelectionSort(Array A, int n)
	FOR k=0 TO n-2                                         ┐
		m <- k+1                                           |
		FOR j=k+2 TO n                ┐                    | 
			IF A[j]<A[m]              | n-(k-2)+1 = n-k-1  | n-1
				m <- j                ┘                    |  
		scambia A[k+1] e A[m]                              ┘
return A
```

$$
\begin{gathered}
T(n)=\sum\limits_{k=0}^{n-2}n-k-1\quad\rightarrow \text{cambio di variabile}\begin{cases}
j=n-k-1\\
k=0 \rightarrow j=n-1\\
k=n-2 \rightarrow j=1
\end{cases}\Rightarrow\\
\Rightarrow\sum\limits_{j=1}^{n-1}j=\frac{n(n-1)}2=\frac12(n^2-n)
\end{gathered}
$$

$$
T(n)=\Theta(n^2)
$$

### Insertion Sort - $O(n^2)$

```jsx
function InsertionSort(Array A, int n)
	FOR k <- 1 TO n-1                           ┐
		e <- A[k+1]                             |
		FOR j=1 TO k+1                 ┐        |
			IF A[j] > e THEN           | k+1    | n-1
				BREAK                  ┘        |
		IF j<k+1 THEN                           |
			FOR t <- k DOWN TO j                |
				A[t+1] <- A[t]                  |
		A[j] <- e                               ┘
return A 
```

$$
\sum\limits_{k=1}^{n-1}k+1\Rightarrow 
\begin{cases}
j=k+1\\
k=1\rightarrow j=2\\
k=n-1\rightarrow j=n
\end{cases}
\Rightarrow\sum\limits_{j=2}^nj=\frac{n(n-1)}{2}-1=\frac12(n^2-n-2)
$$

$$
T(n)=\Theta(n^2)
$$

### Bubble Sort - $O(n^2)$

```jsx
function BubbleSort(Array A, int n)
	FOR i=1 TO n-1                                   ┐ 
		scambia <- false                             |
		FOR j=2 TO n-i+1                   ┐         |
			IF A[j-1] > A[j] THEN          |  n-i    |  n-1
				scambia A[j-1] e A[j]      |         |
				scambia <- true            ┘         |
		IF NOT scambia THEN                          |
			break                                    ┘
	return A
```

$$
\sum\limits_{i=1}^{n-1}n-i\Rightarrow\begin{cases}
j=n-i\\
i=1\rightarrow j=n-1\\
i=n-1\rightarrow j=1
\end{cases}\Rightarrow\sum\limits_{j=1}^{n-1}j=\frac{n(n-1)}{2}=\Theta(n^2)
$$

### Correttezza Bubble Sort

Dopo l’$i$-esima iterazione, gli elementi $A[n-i+1]\cdots A[n]$ sono correttamente ordinati e posizionati

$$
\forall\ h,k,\text{con }1\le h\le k\text{ e }n-i+1\le k\le n\Rightarrow A[h]\le A[k]
$$

Dimostrazione per induzione

BASE $i=1$      il max(A) si trova in $A[n]$

PASSO     supponiamo vero per $i-1$

per l’ipotesi induttiva, $\forall\ h\le n-i+2$ risulta $A[h]\le A[n-i+2]$

dopo l’$i$-esima iterazione, il $max(A[1...n-i+1])$ si trova in $A[n-i+1]$

$\forall\ h\le n-i+1$ risulta $A[h]\le A[n-i+1]$
