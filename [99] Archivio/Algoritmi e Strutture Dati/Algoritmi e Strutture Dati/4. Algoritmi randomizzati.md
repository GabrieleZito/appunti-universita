
Un algoritmo è randomizzato se è determinato non solo dall’input ma anche da un generatore di numeri casuali

In questi algoritmi si calcola il tempo atteso

$$
E[t(I]=T_{exp}(I)=\sum\limits_{\text{sequenza R}}(P(R)*t(I,R))
$$

Nel caso peggiore, su istanze di dimensione n

$$
T_{exp}(n)=\max\limits_{I(n)}\sum(P(R)*t(I,R))
$$

### Ricerca randomizzata

```jsx
FUNCT RAND(L, x)
	permuta casualmente L
	for all y in L do
		if y=x then
			return true
	return false
```

Conviene quando la permutazione costa meno del confronto. Allontana il caso peggiore
