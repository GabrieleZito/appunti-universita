### Funzioni

Il passaggio dei parametri è per valore

```jsx
function nome(paramtero1, parametro2, ...){
	codice
	return risultato        //se return non è presente ritorna void
}
```

Una funzione è un oggetto di tipo *Function,*  quindi può essere dichiarato anche 

```jsx
var nome = function(param, ...){codice}
```

Funzioni arrow

```jsx
var nome = (param, ...) => {codice}
```

### Oggetti

```jsx
var nomeOggetto = {
	prop: valore,
	prop2: valore2,
	metodo: function(param, ...){codice}
}
```

Gli oggetti definiti dall’utente sono di classe Object

Accesso agli elementi:

```jsx
nomeOggetto["prop"];
nomeOggetto.prop;
nomeOggetto.metodo(param, ...);
```

### Dichiarazione di una classe

La dichiarazione di una classe risiede nella definizione della proprietà *prototype* della classe stessa, al cui interno ci sono le proprietà che sono invariate per tutte le istanze della classe

I prototipi si possono creare in vari modi

- Dichiarazione esplicita della funzione costruttore
    
    ```jsx
    function People(name, ...args){
    	this.name = name;
    	this.surname = (args && typeof args[0] == 'string' && args[0]) || null;
    	this.age = (args && typeof args[0] == 'number' && args[0]) || (args && typeof args[1] == 'number' && args[1]) || null; 
    }
    
    People.prototype.isALiar = function(){return false;}
    let Jack = new Peaple('Jack', 'London', 87);
    ```
    
- Metodo Object.create()
    
    ```jsx
    const protoPeople = {
    	name: 'Fred',
    	surname: 'Flintstone',
    	salutation: function(){
    		console.log(`Hi mi chiamo ${this.name} ${this.surname}`)
    	}
    }
    
    let barney = Object.create(protoPeople)
    barney.name = "Barney";
    barney.surname = "Rumble"
    ```
    
- Costrutto class
    
    ```jsx
    class Persona{
    	constructor(name, surname){
    		this.name = name;
    		this.surname = surname;
    	}
    
    	sayHello = function(){
    		console.log(`Hi mi chiamo ${this.name} ${this.surname}`)
    	}
    }
    
    let fred = new Persona("Fred", "Flintstone")
    ```
    

### Polimorfismo

Un codice polimorfo è tale per cui, stabilita una certa interfaccia per i dati su cui opera, lavora in maniera trasparente con tutto ciò che espone quell’interfaccia

### Iteratori

```jsx
var nomeIteratore = oggettoIterabile[Symbol.iterator]();

nomeIteratore.next();
```

Per una classe un iteratore custom si definisce all’interno del protitipo

```jsx
MyClass.prototype[Symbol.iterator] = function(){
	return new myIterator(this)
}
```

### Get, Set e Static

```jsx
class Temperature{
	constructor(celsius){
		this.celsius = celsius;
	}
	
	get fahrenheit(){
		return this.celsius*1.8+32
	}

	set fahrenheit(value){
		this.celsius = (value-32)/1.8
	}
}

let temp = new Temperature(25)
Object.defineProperty(temp, 'celsius', {   //rende riservate le rappresentazioni interne dei dati
	writable: false,
	enumerable: false
})
```

### Ereditarietà

Stabilisce che la nuova classe non ha il prototipo di Object ma quello della classe da cui eredita

```jsx
class ClasseDerivata extends ClasseBase{
	constructor(x,y, a){
		super(x,y);
		this.alpha = a;
	}
}
```
