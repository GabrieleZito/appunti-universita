### Tag META

Contiene informazioni per la corretta gestione della pagina

```html
<meta charset='UTF-8'>    //set di caratteri
<meta http-equiv='X-UA-Compatible' content='IE-Edge'> //compatibilità solo con edge
<meta name="viewport" content="width=devide-width, initial-scale=1">  // informazioni aggiuntive come fossero impostazioni interne del browser

<meta name='robots' content='noindex, nofollow'> //direttive per indicizzazione della pagina (index/noindex) e delle pagine collegate (follow/nofollow)
<meta name='keywords' content='...'>   // lista di parole chiave per indicizzazione
<meta name='description' content='...'>  //descrizione estesa
```

### Tag LINK

Collegamento a risorsa esterna

```html
<link rel='stylesheet' type='text/css' media='screen' href='main.css'>
<link rel='icon' sizes='32x32 64x64' href='favicon.ico'>
```

rel: indica la relazione della risorsa con il documento

media: media query di applicazione del foglio di stile (screen, print, aural, braille, …)

### Tag SCRIPT

Collegamento a script esterno e area di inserimento di codice all’interno del documento (JS default)

```html
<script src='main.js'></script>
```

### Layout

Esistono due tipologie base di elementi in un layout HTML:

- Elementi Block : rettangoli di testo che iniziano e finiscono con un’andata a capo e controllano i propri margini esterni ed il proprio bordo
- Elementi Inline : si inseriscono nel flusso del testo, non vanno a capo e non controllano nè margini nè bordo

### Tag IMG

Elemento inline

```html
<img src='img.png' alt="testo alternativo" title='tooltip' width="150">
```

### Tag A

```html
<a href="www.google.it" target='blank' rel='noreferrer' title='link a google'> testo</a>
```

href : URL di destinazione

target : frame di destinazione (blank, _self, _top, _parent)

rel : noreferrer → non invia informazioni alla pagina di provenienza
