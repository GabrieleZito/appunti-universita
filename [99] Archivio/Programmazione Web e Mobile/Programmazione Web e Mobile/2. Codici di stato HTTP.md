- 1XX : Informational. Richiesta ricevuta, continua l’elaborazione
    - 100 Continue
- 2XX : Success.
    - 200 OK
- 3XX : Redirection. Il client deve eseguire ulteriori azioni per soddisfare la richiesta
    - 301 Moved Permanently - Questa e tutte le future richieste andranno dirette ad un altro URL
    - 304 Not Modified - Non c’è bisogno di ritrasmettere la risorsa: è una redirezione implicita alla cache
- 4XX : Client Error. La richiesta è sintatticamente scorretta o non può essere soddisfatta
    - 400 Bad Request   -  Richiesta non soddisfacibile per errori di sintassi
    - 401 Unauthorized
    - 403 Forbidden    -   La richiesta è legittima ma il server si rifiuta di soddisfarla
    - 404 Not Found   -  La risorsa richiesta non è stata trovata
    - 405 Not Allowed  -  La richiesta è stata eseguita usando un metodo non permesso
- 5XX : Server error. Il server ha fallito nel soddisfare una richiesta apparentemente valida
    - 500 Internal Server Error - Messaggio di errore generico
    - 503 Service Unavailable  -  Server al momento non disponibile
