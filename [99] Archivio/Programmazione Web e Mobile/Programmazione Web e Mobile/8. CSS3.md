Cascading Style Sheets

### Regole di applicazione delle proprietà:

- Specificità: Una proprietà più specifica prevale su una meno specifica
- Ereditarietà: Una proprietà si propaga dall’elemento per cui è definita a tutti i suoi discendenti
- Cascata: Una proprietà definita per un elemento viene sovrascritta da una sua nuova definizione incontrata successivamente

### Selettori

- Selettori di elementi: applicano le regole a singoli elementi o a gruppi selezionati nel dom
- Selettori di attributi: applicano le regole sulla base dei valori degli attributi degli elementi
- Pseudi classi: raggruppano elementi che si trovano in “situazioni” simili rispetto alla loro posizione nel DOM o all’evento che si scatena in essi
- Pseudo elementi: parti caratteristiche di elementi, considerate come se fossero elementi a se stanti

### Elementi semantici

elementi evocativi di quello che potrebbe essere un layout, non hanno proprietà stilistiche e sono degli elementi block

- `<figure>` - contiene immagine con legenda
- `<figlegend>` - contiene la legenda
- `<details>` - dettagli aggiuntivi di contenuto eventualmente da nascondere
- `<summary>` - intestazione visibile per un elemento *details*
- `<main>` - contenuto principale di un documento
- `<mark>` - è un blocco da selezionare e mettere in risalto
- `<time>` - contiene una data/ora

### Form

Raccolta di elementi *input*  dei dati e di *label* ad essi correlate

```html
<form method='post' action='...'>
	<label for='inputid'> Etichetta </label>
	<input id='inputid' name='inputname' type='...'>
</form>
```

### Direttive di preprocessore (at-rules)

Utilizzate per definire dei comportamenti globali nell’applicazione dello stile 

```css
@charset 'utf-8';     //definisce il set di caratteri usato

@import url('custom.css') print;  // importa un foglio di stile
@import 'common.css';             // va inserito dopo charset e prima di qualsiasi altra regola

@namespace url(http://www.w3.org/1999/xhtml);  // indica il namespace di riferimento per l'utilizzo di selettori non standard

```

Regole annidate

```css
@font-face{
	font-family: ItalianBreakfast;
	src: url(multimedia/ItalianBreakfast-Regular.otf) format("opentype");
}
```

```css
@supports (display:grid){     // include le regole in maniera condizionale a seconda delle proprietà supportate dal browser
	div{
		display: grid;
	}
}

@supports not (display:grid){     
	div{
		float: right;
	}
}
```

```css
@page {       // attiva solo alcune regole quando si stampa la pagina
	margin: 1cm;
}

@page :first{
	margin: 2cm;	
}
```
