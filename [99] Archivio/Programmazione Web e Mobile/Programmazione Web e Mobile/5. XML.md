### Tipi di marcatura

- Procedurale - Descrive come processare il documento (postscript, rtf)
- Descrittiva - Descrive la struttura logica del documento (HTML, SGML, XML)

### SGML - Standard Generalized Markup Language

Metalinguaggio di marcatura. Permette di definire linguaggi di marcatura

- Estremamente espressivo e configurabile → elaborazione del testo complicata
- Manca della gestione di link e del conflitto sui nomi delle etichette
- I documenti devono essere validi e ben formati

### HTML - HyperText Markup Language

- Linguaggio di marcatura
- La marcatura denota il suo formato, non il significato(struttura logica)
- Studiato per il web

## XML - eXtensible Markup Language

- Evoluzione di HTML
- Marcatura dettata dalla struttura logica
- Document Type Definition - permette di dichiarare la struttura che tutti i documenti di un certo tipo devono rispettare

### Caratteristiche di XML

- Marcatura dichiarativa - etichette che indicano la funzione astratta del testo a cui si riferiscono
- Marcatura strutturata - permette di raggruppare parti del testo come unità strutturali complesse
- Marcatura gerarchica - Le strutture possono combinarsi in maniera gerarchica
- Ogni elemento è marcato da un tag di apertura e uno di chiusura (è case sensitive)
- I Tag:
    - possono contenere solo lettere, cifre e . - _
    - possono cominciare solo con lettere o _
- Ci deve essere un tag radice che contiene tutto
