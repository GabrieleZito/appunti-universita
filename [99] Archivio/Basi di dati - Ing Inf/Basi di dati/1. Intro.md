
Sistema informativo → Sistema che permette lo scambio di informazioni tra diversi attori.

Sistema informatico → Parte automatizzata del sistema informativo.

Dato → Ciò che è immediatamente presente alla conoscenza, senza elaborazioni.

Informazione → Insieme di dati o elemento che permette di avere conoscenza su una situazione o fatto.

Database → Insieme organizzato di dati utilizzati per il supporto allo svolgimento delle attività di un ente (generica). 

Database → Insieme di dati gestito da un DBMS (specifica).

DBMS → DataBase Managment system. Gestisce collezioni di dati Grandi, Persistenti, Condivise. Garantisce Privatezza, Affidabilità, Efficienza, Efficacia.

Le operazioni su un DB sono atomiche. Cioè devono essere eseguite per intero per poter essere salvate.

Un DB permette l'indipendenza dei dati, cioè il dato non dipende dalla struttura fisica realizzata.
